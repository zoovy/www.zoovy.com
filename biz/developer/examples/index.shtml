Example Code

orderdispatch.pl.txt: a sample script which pulls in and interprets the contents of an order via order dispatch.

shipquote.cgi: a sample of how to use the Zoovy WebAPI to perform realtime shipping quotes. Ideal for Auction users who want to provide shipping quotes by zip code.

framethis.cgi: a quick example which describes how to redirect a user into frames, and then handing off control to the auction claim page. Written in Perl.

zexport-1.11.tgz: a content replacement system written in Perl which faciliates easy integration of the Zoovy product database (and product manager) with custom Zoovy developer websites. Effectively allowing anybody to use the Zoovy Product Management system with their own custom website.

Paypal Cart Conversion Directions - Zoovy has developed a tool to make it easy for customers who are migrating from a paypal cart to Zoovy.

createexternal.pl - a perl script which will create an external (incomplete) item in the Zoovy system.

uploadimg.zip - Delphi Code to upload an image to the Image Library using RFC 1867 and Indy SSL Components.
