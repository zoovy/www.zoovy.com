Site Integrator Configuration
Zoovy WebAPI
Provides XML methods for retrieving and storing virtually any piece of data in a Zoovy store.
URL: http://www.zoovy.com/webapi
Shipping API
Allows the Zoovy store to query a remote CGI/ASP/PHP/JSP program which returns shipping rates.
To Access: Click on Setup | Shipping | External API Configuration
Order Dispatching
Facilities the transmission of orders to a remote system when they are received, or paid for.
To Configure: http://www.zoovy.com/biz/setup/dispatch/
Promotion API
Transmits the contents of the cart to a remote program for the purpose of computing discounts, adding promotions, or some other user defined behavior. Allows a remote program to completely "rewrite" the contents of the cart.
Promotion API URL: Documentation
Promotion API attributes: comma separated list of attributes to be passed.
