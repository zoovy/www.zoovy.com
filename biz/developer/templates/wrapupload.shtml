Comment Tag Descriptions

<-- HEAD -->
This comment tag should appear between the <head> and </head> tags that you add into your HTML code. The comment adds in the proper title and meta tags for keywords and descriptions as defined for the page in the Zoovy system.
<-- TITLE -->
This comment tag is replaced with the actual title of the page when the Zoovy system outputs it.
<-- BODY -->
This is replaced with the contents of the page... the contents of the page look like whatever wrapper theme you have chosen in the Theme Chooser under the Setup tab. The colors and fonts used in the page contents are all determined by the theme.
