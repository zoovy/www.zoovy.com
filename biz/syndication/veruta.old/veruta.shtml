Veruta Merchant ID:
Your merchant id can be found in a javascript variable provided to you by Veruta. Set this field to blank if you wish to disable Veruta on your site. Once the Merchant ID is set, the Veruta tracking code will be automatically appear on your site. Contrary to the generic instructions provided by Veruta, Zoovy clients do not need to install any javascript to make this integration work.
