eBay Store Syndication Manager

Product Sources
* Keep in mind, regardless of setting - products which are explicitly denied from launching to eBay will NEVER be included in syndication.
name="ENABLE_AUTOPILOT"> Enable AutoPilot Mode [REALLY REALLY RECOMMENDED]
Auto-Pilot allows Zoovy's eBay Store Syndication Engine to make 'reasonable guesses' about getting your products up on eBay. This includes using fields such as the retail price on your website as the eBay Item Fixed price (so the price on your website will be the same as the price on eBay). Without Auto-Pilot, you will need to configure eBay settings per product, if settings are configured the product will generate errors and you will either need to use the debug feature in syndication OR disable the product from syndication to make the error go away. If you receive too many errors, your token will be deactivated and your eBay processing will be suspended.
name="DUPLICATES"> Allow Duplicates [NOT RECOMMENDED]
What this does: This setting looks across all listings which are already on eBay via powerlister, or channels via syndication in the same account/partition. If disabled then items which are already on eBay (in any form, including auctions) will not be included. ** Keep in mind that powerlister is entirely autonomous from the syndication engine - and it doesn't necessarily re-launch items immediately so there is absolutely a chance that a fixed price listing could be listed concurrently (between the time one listing ends and powerlister launches another), however this setting will make a good-faith effort to try and prevent that. If this good-faith effort is unacceptable, then you will want to disable items from syndication when launching them either manually or with powerlister.
name="LOGGING"> Detailed Logging [NOT RECOMMENDED]
What this does: The syndication engine generates detailed log files (which can be obtained by asking support), that include the decision logic that was used to perform a listing. These log files consume significantly more disk space. You should probably only enable this if instructed to by Zoovy support.
name="SUSPEND_AFTER_ERR"> Suspend items with errors + Add to To-Do list [RECOMMENDED]
What this does: When enabled, when a product receives a launch error as a result of a listing event created by syndication, it will disable the product from future launches, set the reason to the error, and create an entry in your to-do list to correct the issue. Enable this setting when your token is receiving too many errors is being disabled.
Max Syndication Listings:
How this works: use -1 for unlimited, or specify the maximum number of fixed price syndicated listings you'd ever want to have up on eBay. In addition it's worth mentioning that Products which have options, or errors will *NOT* be syndicated however they consume a spot in line, and therefore the maximum number may never be achieved unless those products are excluded from syndication.
Default Launch Profile:
This launch profile will be used if none is configured. If you use partitions, and a product is associated with a launch profile on a different partition - THEN - that product cannot be syndicated on another partition, unless it has a separate profile configured for that partition using a flexedit field.
