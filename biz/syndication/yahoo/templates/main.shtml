Yahoo Store Inventory Synchronization
This is a feature for people who use Yahoo stores in conjunction with Zoovy stores. First Enable the feature, then setup your Yahoo store to pull real time inventory from the URL: http://www.zoovy.com/webapi/yahoo/inventory.cgi/YOURNAME make sure you replace YOURNAME with the username of your Zoovy store. For information on how to setup Real-time inventory for a Yahoo store see the documentation on the Yahoo site, just do a search for "real-time inventory" in help.

Yahoo Inventory:
name='yahoo_sync'>Allow Yahoo Stores to Synchronize with Zoovy.
 Yahoo Store Order Synchronization
name='yahoo_ordersync'> Allow transmission of orders from Yahoo store to Zoovy.
name='yahoo_createcustomer'> Create Customer Records.
name='yahoo_optin'> Opt Customers into Mailing List.
name='yahoo_process'> Process incoming credit cards. [do not enable this if your Yahoo store also processes credit cards.]
name='yahoo_notify'> Send order notification to customers.

Order ID Behavior:

Please select a synchronization password using only upper and lower case letters and numbers, 10 digits or better is recommended.
Synchronization Password:

Synchronization URL: https://webapi.zoovy.com/webapi/yahoo/ordersync.cgi/pass=PASSWORD/user=
Be sure to replace PASSWORD with your Yahoo synchronization password from above.
