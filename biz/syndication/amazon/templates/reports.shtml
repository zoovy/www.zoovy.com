Settlement Reports:

These settlement reports are tab-delimited text files. These can be difficult to read in a text editor such as Notepad (on a Windows PC) or Text Edit (on a Macintosh), but you can easily open them in most spreadsheet applications (such as Microsoft Excel) or database software.

The easiest way to read settlement reports is to save them to your computer and then open them in your favorite spreadsheet or database application. Then you can easily summarize, manipulate, or reformat the data the way you want.

Please note:
* Some web browsers automatically open settlement reports with your computer's default text editor. If this happens, use the File menu to save a copy of the report to your computer before opening it in another application.
* On a Windows PC, you can right-click the link for the settlement report to save the file to your computer.

For more information about what each column means in a settlement report, as well as tips and tricks for reading the file in Microsoft Excel, please see Amazon Settlement Reports.
