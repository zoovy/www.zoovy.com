Desktop Software
Zoovy Integrated Desktop contains three applications:
  1. Order Manager
    Easily manage your orders and customers. The order manager will download any orders placed through your Zoovy store, automatically send customers email, manage tracking numbers, UPS/FedEx/USPS (Endicia) airbill printing, Quickbooks integration, advanced printing support and so much more!
  2. Warehouse Manager
    Makes the day to day tasks of managing products, images, and categories easier and faster than ever.

Network Database Installation/Deployment
Zoovy supports network database installations for customers who need concurrent user access. Please download and install the following application on your network database server:
  • Zoovy Network Sync Manager
    [[Download v]]
    A windows service that automatically synchronizes your local databases with Zoovy servers. This is intended for clients running network databases and manages the synchronization process automatically.
If you do not currently have a supported database installed on your server, you will also need to select a database that fits your network environment. A discussion of network databases, and deployment options along with frequently asked questions is located in: webdoc #51344.
  • MySQL 5.1 Community Server
    • MySQL 5.1 Community Edition
      A high performance free database, recommended!
    • MySQL 5.2 Connector/Net
      Necessary for the server, and must be installed on each client as well.
  • Microsoft SQL Server 2008
    • Microsoft SQL Server 2008
      The free version of SQL Server supports up to 4gb databases and 1gb of memory.

First Time Installation Notes
Before installing any of the applications above you should consider the following:
  • .Net Framework Required
    All applications require the Windows Installer 3.1, and Microsoft .NET 3.5 sp1 framework to be installed for proper operation. Please use the links to download:
    Windows Installer 4.5
    Microsoft .Net 3.5 sp1 Download Site
  • Network Database Installation
    Installation documentation can be found: Webdoc #50491.
    Warning: Network Database installations are difficult multi-step processes with lots of room for mistakes. We encourage companies who do not employ a fulltime database administrator (which is different than a "Tech Guy") to consult with our Geek Services to deploy your database. In order to speed up the installation appointment, please view our Webdoc #51357 prior to your appointment.
  • Multi-Site Considerations
    Only one database may be active per account. If you have multiple sites or remote users you will need to use a VPN solution to connect them.
    Zoovy Geek Services can assist you with multi-site VPN deployments.
  • Enterprise Support
    Zoovy provides an enhanced level of support for our larger clients who are more sensitive to service disruptions. Enterprise support is also suitable for clients who need specific customizations made to their software version. Enterprise support also offers scheduled upgrades on weekends and off-business hours to minimize potential disruption to business operations. Please contact billing@zoovy.com to determine eligibility for this service.
  • Supported Operating Systems:
    Windows XP sp3, Vista sp2, Windows 7, Windows Server 2003 r2; Windows Server 2008. Requires .NET Framework 3.0.
    64 Bit Considerations: Order Manager will run correctly on Vista/Windows 7 64 Bit Operating System. Other applications such as Quickbooks and Endicia will not integrate properly. In addition various peripherals such as many models of HP Printers are known to have issues and will cause the Order Manager software to become unstable.
