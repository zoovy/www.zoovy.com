Windows XP Installation Instructions

There is feature in Windows XP that causes the system to cache the old exe. Our development team is currently working on a solution to solve this problem. In the meantime, instructions on how to fix this are listed below.

For Order Manager:

  1. Uninstall the Order Manager
  2. Delete the orderedit.exe typically located in C:\Program Files\Zoovy\Ordermgr
  3. Reboot the computer
  4. Installed the new version of the Order Manager

For Product Manager:

  1. Uninstall the Product Manager
  2. Delete the prodmgr.exe typically located in C:\Program Files\Zoovy\Prodmgr
  3. Reboot the computer
  4. Installed the new version of the Product Manager

Any questions, please feel free to contact Zoovy Technical Support by email at support@zoovy.com or by filling out a trouble ticket at http://support.zoovy.com. We thank you for your patience and understanding.
