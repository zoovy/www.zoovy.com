name="HOST_TYPE" value='DEFAULT'> Default/Primary Domain
    All pages will be hosted under the domain name specified, uses the DEFAULT profile.
name="HOST_TYPE" value='SPECIALTY'> Specialty Site Mode - (requires Cross selling bundle access)
    Allows for a different wrapper, logo, and company information for this domain.
name="HOST_TYPE" value='REDIR'> Redirect Domain
    Redirects all traffic for this domain to another domain when a visitor lands on the page. Frequently used for parked/placeholder domains e.g. yourname.net redirects yourname.com
