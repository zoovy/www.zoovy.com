Load Products from CSV File


Advanced Product - use the first line of your CSV file to specify which attributes will be imported.
    Allow CR/LF in file (WARNING: you must make sure all fields are properly escaped/quoted to use this).
    Do not Encode Categories, they are already encoded as safe names (normally disabled).
    Configure Management Categories using Navigation Categories schema.
    Remote System Synchronization: Delete products which were are not found in this import.
    Reorder navigation categories after import. (only necessary if updating category sort fields via update - VERY SLOW)
    Perform Non-Destructive Import - updates products, rather than doing a complete overwrite of a products data! (slower)
