Advanced Tax Rules



Usage: This file can be exported into a text editor, or saved as a CSV file and imported into a spreadsheet.

The columns are: Rule type, Match value, Tax Rate, Applies to.

The Applies To value is a bitwise value, it is the sum of the following values: enable=1, shipping=2, handling=4, insurance=8, specialty fees=16. For example to enable tax on shipping+handling you would add 1+2+4 and set the value to "7", subtotal is always implied as part of the tax value.
