  Fixed Shipping Calculator: Single External Items
Explanation: This shipping calculator is only used when one External (e.g. Auction) Item appears in the cart, in addition the External Item quantity must be equal to 1.

Method Name: (Letters A-z,0-9 and spaces only)

Behavior:
value="0"> Disable This Shipping Method (use other methods).
value="1"> Use this Method with other Calculated Methods.
value="2"> Disable All Other Shipping Methods

Calculation Method:
value="0"> Use Fixed Shipping Cost.
value="2"> Add Fixed Shipping Cost to Actual Computed Shipping Cost(s).

  Fixed Shipping Calculator: Multiple External Items
Explanation: This shipping calculator is only used when multiple External (e.g. Auction) Items appears in the cart, for example a winner has won two or more different items OR the winning bid quantity was greater than one.

Method Name: (Letters A-z,0-9 and spaces only)

Behavior:
value="0"> Disable This Shipping Method (use other methods).
value="1"> Use this Method with other Calculated Methods.
value="2"> Disable All Other Shipping Methods

Calculation Method:
value="0"> Combine All Shipping Costs.
value="1"> Only Apply Most Expensive Shipping Cost one time.
value="2"> Add Only the Highest Fixed Shipping Cost to Actual Computed Shipping Cost(s).
value="3"> Add Any Fixed Shipping Costs to Actual Computed Shipping Cost.
