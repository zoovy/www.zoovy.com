name="limiteduse"> Newsletter/Limited Use
When enabled this coupon may only be added to the cart using a limited use link. Limited use links are automatically generated when a newsletter is sent out, but could also be generated programatically by custom applications. If this is enabled the coupon MAY NOT be added directly to the cart from "Coupon Code" box.
name="stackable"> Allow Coupon Stacking/Combinations.
How this works: Coupons codes are added to the cart in alphanumeric order based on coupon code. Each non-stackable coupon will remove all other coupons (which may have been added before it). When stacking is enabled, any non-combinable codes will need to be explicitly removed using rules. If you're not sure what to do here, leave this disabled.
