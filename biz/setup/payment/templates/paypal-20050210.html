Address Verification
Paypal will only insure funds for packages which are sent to a PayPal confirmed address. Unfortunately many buyers provide Zoovy with a non-PayPal Confirmed, alternative address for shipping. Zoovy can flag orders which are not being shipped to a verified address and put them in payment status 'REVIEW' instead of payment status 'PAID'.

  eBay/Paypal Checkout Features
PAYPAL IPN ORDER CREATION
Your Paypal.com account has a feature called "Instant Payment Notifications" (IPN for short) which sends a notification of each payment to a remote program such as Zoovy. Zoovy can use these payment notifications to automatically create orders for customers who did not use Zoovy Checkout to pay for their order.
  1. Login to your Paypal account
  2. Go to Profile, then select Selling Preferences
  3. Select Instant payment Notification
  4. Click 'edit link' and place the URL "http://paypal.zoovy.com/webapi/paypal/ipn.pl/" (without the quotes) into the space provided.

Configuration
> Correlate exact and reasonable matches [RECOMMEND: Yes]
> Notify me of unknown payments [RECOMMEND: Yes]
> Attempt to correlate unknown payments. [RECOMMEND: No]
> Assume everything is a Chinese Listing [RECOMMEND: No]
> Create phantom orders for unknown items [RECOMMEND: No]
> DO NOT Automatically change orders shipping location to the confirmed address [RECOMMEND: No]
> Audit shipping [RECOMMEND: No]
> Process promotions [RECOMMEND: No]

> Allow PayPal Checkout from Cart Page [RECOMMEND: Yes]
This allows your customers to bypass the Zoovy checkout and use PayPal's checkout instead. Shipping through the PayPal system will be sent as whatever the quoted shipping is in the cart, so you should enable "Cart Quote Shipping" in the Checkout Properties area of Setup. Note that we currently do not ask for the country in the cart so enabling this is not recommended if you ship international orders (they will be quoted as $0.00 shipping (Actual cost to be determined). Also note that promotional rules which create discounts in the cart will also not work with this feature... the PayPal system does not allow Zoovy to send negative-priced shopping cart items to thier system. YOU MUST ENABLE THIS FEATURE, ALONG WITH THE IPN FEATURE IF YOU WANT TO BUYERS TO QUALIFY FOR PAYPAL INSURANCE, OR YOU ONLY WANT TO ACCEPT PAYMENTS FROM VERIFIED ADDRESSES.
