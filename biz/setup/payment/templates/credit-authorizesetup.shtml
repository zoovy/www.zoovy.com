API LOGIN: API Login is NOT the same as your authorize.net merchant login. It is a separate login and password provided by going to Account | Settings | API Login ID and Transaction Key.

HINT ABOUT TRANSACTION KEYS: You must be configured to be on "LIVE" - not in "TEST" mode. In addition you must re-issue your Transaction Key on the LIVE mode, any transaction keys issued while in "TEST" mode are not valid in "LIVE mode.
