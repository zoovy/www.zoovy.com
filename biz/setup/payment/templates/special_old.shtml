EXPERIMENTAL FEATURES
Paypal Payment Correlation
The correlation feature automatically creates orders in your store from buyers who pay via paypal but do not go through checkout.
How it works: By matching on email addresses Zoovy can intelligently flag external items as paid, then instantly create orders, customer accounts, and customer passwords. Orders created via correlation can automatically be managed and synchronized with both order manager, and windows order manager.

name='PAYPAL_CORRELATE' value='0'> Disabled - only handle transactions through the Zoovy store.
name='PAYPAL_CORRELATE' value='1'> Simple: Enable Correlation, don't attempt to match shipping - assume customer is always paying the correct amount (safe).
name='PAYPAL_CORRELATE' value='10'> Advanced: Enable Correlation, combines external items attempts to match minimum shipping, notifies merchant if error occurs (leaves order unpaid).

name='PAYPAL_FEEDBACK'> Encourage checkouts by only leaving feedback for Paypal payments received through Zoovy store.

To obtain your MoneyZap Customer Number:
  1. Login to your Money Zap Account,
  2. On the left hand toolbar click the option which says "MONEYZAP TOOLS".
  3. In the MoneyZap Tools window, click the moneyzap buttons tab.
  4. On the next page leave Amount, Description, and Reference number fields blank, then click the Continue button.
  5. The next page will show some HTML code in a window, the 4th line in the window will look like:
    <input type="hidden" name="MZUserID" value="####">
  6. Whatever number appears in the value="###" is your Money Zap customer number! [yes, we've asked them to make it easier, if you need help contact Zoovy technical support.]
