Paypal Website Payments Pro Configuration
Website Payments Pro requires a separate subscription from Paypal. Unlike Paypal Express Checkout which lets you accept credit cards via Paypal's site for free, this is a true 'merchant account' which lets you accept credit cards directly on your site.

Requirements
  1. If you use Paypal Website Payments Pro you MUST have Paypal Express Checkout enabled. Website Payments Pro uses the Paypal Express Checkout settings to communicate with Paypal.
  2. In order for Website Payments Pro to work your Paypal account must be authorized for the both Website Payments Pro and the Paypal "Virtual Terminal" product.
  3. In addition you will need to have provided the appropriate authorizations to Zoovy's 3rd party developer key: paypal_api1.zoovy.com
