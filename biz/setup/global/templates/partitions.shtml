Some things to remember:
  • Partitioned Data must be managed separately.
  • Because changes are permanent you *MUST* consult with Zoovy Implementation before utilizing this feature. Any changes to a partition once it has been created will result in billable charges.
  • Non-partitioned settings are shared with the default profile (and are NOT configurable by partitioned users).
  • Additional monthly fees apply for each unique data partition.
  • If system messages are partitioned, it will require one unique set of messages per language (when multiple languages are implemented).
