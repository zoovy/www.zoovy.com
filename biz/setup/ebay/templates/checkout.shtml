name="DO_IMPORT_LISTINGS"> Auto-Sync eBay Account (RECOMMENDED)

You should ENABLE Auto-Sync:
.. if you use Zoovy exclusively to process eBay sales. This will result in the best performance/reliability because Zoovy will be able to automatically correct/recover a variety of common types of errors.
.. if you use another listing tool, and want Zoovy to automatically import eBay listings (and sales) that were not created by Zoovy.
.. if you plan to create Second Chance offers outside of Zoovy (ex: via ebay.com)

You should NOT ENABLE Auto-Sync:
.. if you concurrently use a another listing tool (or ebay.com) to create/manage eBay sales on the same account. Auto-sync will import any eBay sales created regarldess of the origin.
IMPORTANT: Non-Zoovy created eBay listings *MUST* have the SKU set, and a corresponding product id configured in the zoovy account product database in order to create a valid order.
