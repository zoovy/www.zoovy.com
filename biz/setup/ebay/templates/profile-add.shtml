Understanding Profiles:
Company Profiles store information about your company's contact information, policies, and preferences. A single profile can be associated with products, website/domains, and marketplaces:

UsageBenefits
With Products: Products use profiles when launching to a marketplace to select which template, company information and preferences to use. Profiles make it easier to update policies in future and to make changes to existing listings. (Using the "Refresh Listings" button in eBay Manager will cause all your products to be revised with those new policies).
With Marketplaces: Profiles are used to store your listing preferences such as shipping, returns, listing duration, listing template, and other NON-product specific fields. Advanced users may also be interested to know that profile properties can be overridden by products by simply having the same fields set in the product (this is usually done using a customized template) -- doing this can dramatically reduce the number of profiles necessary to operate a business.
Common Profile Gotchas:
  • Each profile is assigned a unique 8 digit code that you will use to refer to it. Devising a naming plan is useful if you intend to have many profiles.
  • Each account has one profile named DEFAULT that cannot be deleted (non speciality domains will always use this profile)
  • The company properties (logos, emails, contact info) used in a profile can be accessed through the website builder only if a profile is associated with one or more specialty domains.
  • The company properties can also be edited by editing the profile properties of a eBay listing template.
  • Changing the company properties of a profile affects all speciality sites, and listings using that profile.
  • Products will always remember the last profile they are used with. Refreshing a listing will be done with the most recently used profile for the product.
  • If a profile is deleted while it is still being referenced then the DEFAULT profile will be used. Avoid deleteing profiles if they are still in use, this can lead to unpredictable system behaviors.

Hint: Getting started with profiles can be a little intimidating, don't hesitate to ask for assistance from support!
