Snapshot Type:
Snapshots are copies of a partition whereas full site restores/backups include data from all partitions. Snapshots are replicated to each webserver in a cluster, and copied on the local disk. Because all persistent data on our site is kept on the local disk, it can be accessed much faster by the webservers. It also allows you to make changes to your site without those changes appearing instantly. This functionality is currently considered EXPERIMENTAL and should only be used at the recommendation of Zoovy technical support.
