Step 2: Specify Export Columns
Analyze Fields:
Click the button to display which fields are used in the selected products.
-OR- specify your own custom fields:

TIP: Leave "Fields" blank to export a full product record with dynamic headers. To specify header fields, specify product attributes separated by commas or newslines, leave no whitespace e.g. "zoovy:prod_name,zoovy:base_price,zoovy:prod_image1" (without the quotes) - without specifying which fields you want to export you will be limited to 500 products per export.
ADVANCED USERS: You can alternatively specify a map to generate a particular CSV file for a trading partner. Use multiple lines

Step 4: Choose Export Options
name=".navcats"> Export Navigation Categories
name=".categories"> Export Management Categories
name=".imagelib"> Export Image URLs rather than Image Library names (may break Excel)
HINT: remember to use %IMGURL=zoovy:prod_image1 as your header to get the URL instead of the image name.
name=".stripcrlf"> Strip Carriage Return/Line Feeds
name=".convertutfx"> Attempt to convert UTFx data to HTML.
name=".produrls"> Enable the %PRODUCTURL header.
name=".addtocart"> Enable the %ADDTOCART header.

HINT: Use /Remember Settings/ to store these settings for future exports.
