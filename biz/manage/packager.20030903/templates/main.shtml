Site Packaging Tool: Creates a CSV or Excel export of products suitable for import into another Zoovy site using the advanced CSV import tool.


Special Options:
Export Navigation Categories
Export Management Categories
Export Image URLs rather than Image Library names (may break Excel)
Timestamp

Usage Notes:
If you are using Internet Explorer and have Microsoft Excel installed DO NOT USE THE CSV EXPORT. Excel will not properly import URL's or HTML from a CSV file. To work around this defect in Excel, use the Excel export (which will create a webpage with an Excel 2000 spreadsheet embedded in it) when you are done then save the file as an XLS file. TO IMPORT FROM EXCEL: YOU MUST SAVE THE DATA AS CSV in Excel, to do this please select [File] and [Save As] and change the type to "Text/CSV". If you attempt to re-import the native Excel file that was exported it will absolutely corrupt your product database.

Customers using Netscape, Opera, or Mac OS should save the CSV file and open it in the application of their choice (Filemaker, etc).


Due to limitations of string length this export is known not to work with Excel 5, or Excel 97.
