Usage Notes:
TO IMPORT FROM EXCEL: YOU MUST SAVE THE DATA AS CSV in Excel, to do this please select [File] and [Save As] and change the type to "Text/CSV". If you attempt to re-import the native Excel file that was exported it will absolutely corrupt your product database.

Customers using Netscape, Opera, or Mac OS should save the CSV file and open it in the application of their choice (Filemaker, etc).
