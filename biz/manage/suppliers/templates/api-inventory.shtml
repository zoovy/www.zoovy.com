Directions/Hints:
  • This area is to configure a daily load of inventory from a supplier's public file.
  • Look in the file your supplier automatically generates
  • Map the columns in the file so Zoovy can automatically pick them up
  • The first column is number 1
  • Set columns you do not want to use to zero
  • Only lines which contain a valid SKU will be imported (all other lines will be ignored)
  • Availability can be any form of Yes/No True/False or number (if true the item will be set to unlimited quantities).
  • Inventory will be importly on a nightly basis.
