Simply press the UPDATE button below to manually create order items for your STOCK items that have gone below their minimum quantity. This action is only needed when items have been created with zero (0) inventory and a minimum stock qty greater than zero.

In all other cases, order items are automatically created when inventory is updated via the following methods:
  • customer buys product from your website (or Amazon, ebay, etc) OR
  • csv inventory import OR
  • manually thru the Product Editor
